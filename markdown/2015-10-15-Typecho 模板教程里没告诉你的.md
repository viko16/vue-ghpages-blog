## 循环里的第几个

比如想要第一篇文章特殊处理

``` php
<?php while ($this->next()): ?>
    <?php if ($this->sequence == 1) : ?>
        啦啦啦我是第一篇
    <?php endif; ?>
<?php endwhile; ?>
```

## 人性化时间输出

输出”16小时前“这样的人性化时间

`<?php $this->dateWord(); ?>`

## 按字数生成摘要

无格式的正文摘要，第一个参数为多少个字符，第二个参数为后置字符

`<?php $this->excerpt(250, '...'); ?>`
